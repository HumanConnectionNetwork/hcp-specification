# Code of Conduct

## Our Pledge

The Humanitarian Connection Protocol (HCP) community is committed to creating an open, respectful and inclusive environment for everyone.

We welcome contributors from all backgrounds, cultures, countries and levels of experience.

Our shared mission is larger than any individual organization or contributor: building open standards that improve humanitarian coordination.

---

# Our Values

Members of our community are expected to:

- Treat everyone with respect and empathy.
- Assume good intentions.
- Welcome constructive feedback.
- Share knowledge openly.
- Encourage new contributors.
- Focus on solving problems collaboratively.
- Respect cultural and linguistic diversity.
- Prioritize humanitarian impact over personal recognition.

---

# Examples of Positive Behavior

Examples include:

- Helping newcomers.
- Providing constructive technical feedback.
- Respectfully challenging ideas rather than people.
- Improving documentation.
- Sharing implementation experiences.
- Reporting issues responsibly.
- Supporting open collaboration.

---

# Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment or discrimination of any kind.
- Personal attacks or insults.
- Hate speech.
- Trolling or deliberate disruption.
- Publishing private information without consent.
- Intimidation.
- Abusive language.
- Bad-faith participation.

Technical disagreement is welcome.

Personal hostility is not.

---

# Community Standards

When discussing protocol changes:

- Debate ideas, not people.
- Support arguments with evidence.
- Be open to changing your opinion.
- Keep discussions focused on improving the protocol.

Remember that contributors may come from different technical, cultural and humanitarian backgrounds.

---

# Humanitarian Focus

HCP exists to improve humanitarian coordination.

Whenever possible, contributors should consider:

- Does this improve interoperability?
- Does this reduce complexity?
- Does this benefit humanitarian response?
- Does this respect privacy and human dignity?

These questions should guide our decisions.

---

# Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

They may:

- Request changes in behavior.
- Moderate discussions.
- Remove inappropriate content.
- Temporarily or permanently restrict participation in severe or repeated cases.

Enforcement decisions should always aim to protect the health of the community.

---

# Scope

This Code of Conduct applies to all project spaces, including:

- GitHub repositories
- Issues
- Pull Requests
- Discussions
- Documentation
- Community events
- Official communication channels

---

# Reporting

If you experience or witness behavior that violates this Code of Conduct, please contact the project maintainers through GitHub or the official Human Connection Network communication channels.

Reports will be handled respectfully and, whenever possible, confidentially.

---

# Attribution

This Code of Conduct is inspired by the Contributor Covenant and adapted to reflect the mission and values of the Humanitarian Connection Protocol community.
