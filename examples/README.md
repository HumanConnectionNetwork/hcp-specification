# HCP Reference Examples

This directory contains non-normative examples illustrating how the Humanitarian Connection Protocol may be implemented.

These examples are intended to help developers understand the protocol and validate their own implementations.

Examples may include:

- Canonical Humanitarian Records
- Query Requests
- Query Responses
- Correlation Candidates
- Node API examples
- Synchronization messages

If an example conflicts with a specification document, the specification always takes precedence.
