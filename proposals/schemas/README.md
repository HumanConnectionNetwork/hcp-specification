# HCP Schemas

This directory contains the machine-readable schemas that define the Humanitarian Connection Protocol (HCP).

Schemas are provided to help developers validate, exchange and implement HCP-compatible data structures.

Future schema formats may include:

- JSON Schema
- YAML
- OpenAPI components
- XML (if required)
- Protocol validation rules

These schemas are derived from the HCP Specification and should remain fully compatible with it.

The HCP Specification is the authoritative source for all protocol definitions.
