# Core Consolidation

Status: In Progress

## Objective

This sprint focuses on consolidating the HCP Core Specification.

No new protocol features are introduced.

The objective is to improve consistency, terminology, cross-references, documentation quality and implementation readiness.

## Scope

- Terminology review
- Cross-document consistency
- Cross-reference verification
- Architecture consistency
- Official glossary
- Official examples
- Diagrams
- Conformance checklist

## Out of Scope

- New protocol features
- New Core specifications
- Breaking architectural changes

## Guiding Principles

One concept.
One definition.
One owner.

Semantic consistency over document independence.

Interoperability through shared meaning.

Humanitarian Records represent observations.

Humanitarian Cases represent interpretation.

Synchronization exchanges observations.

Correlation creates understanding.

Human verification determines reality.
